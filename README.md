Análisis de Fondos de Pensiones y Cesantías
Descripción general

Este repositorio contiene un análisis integral de datos de fondos de pensiones y cesantías en Colombia, enfocado en comprender el comportamiento de los fondos a lo largo del tiempo, su composición, rendimiento, variabilidad y relaciones entre distintos indicadores financieros.

El proyecto desarrolla un flujo de trabajo analítico completo, priorizando la calidad de los datos, el análisis exploratorio y la claridad en la interpretación de resultados, con un enfoque reproducible y extensible.

Objetivos

Analizar el desempeño histórico de los fondos de pensiones y cesantías

Estudiar la composición y asignación de activos de los fondos

Explorar la variabilidad de los rendimientos y posibles indicadores de riesgo

Identificar patrones y tendencias temporales mediante análisis de series de tiempo

Generar información útil para el monitoreo financiero y la toma de decisiones

Enfoque analítico

El proyecto sigue un flujo de trabajo estructurado:

Carga de datos desde fuentes históricas

Limpieza y preprocesamiento (manejo de valores faltantes, formatos y normalización)

Construcción y transformación de variables

Análisis exploratorio de datos (EDA) mediante estadística descriptiva

Exploración temporal para analizar tendencias y dinámicas en el tiempo

Visualización de datos para comunicar patrones e insights de forma clara

Herramientas y tecnologías

Python

Pandas – manipulación y transformación de datos

NumPy – análisis numérico

Matplotlib / Seaborn – visualización de datos

Jupyter Notebooks – análisis exploratorio y documentación
